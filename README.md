# Cybersecurity-Labs (Kali / Apporto)

This repository showcases my **cyber security lab work** in a Kali (Apporto) environment, starting from **Linux CLI fundamentals** and progressing to core security topics such as **risk & essentials, cryptography, network security/scanning, vulnerability scanning, cloud security, and penetration testing**.

Each week includes **evidence** such as a **DOCX report, screenshots, and exported artifacts** to demonstrate reproducible, hands-on practice aligned with the module structure.

## Security Context

The lab series aligns with core cyber security pillars and practices emphasised in the module:

- **CIA Triad** (**Confidentiality, Integrity, Availability**) as a lens for threats and controls.
- Progressive exposure to **threats** such as malware, SQL injection, DDoS, and MITM, alongside **controls** such as strong authentication, updates, firewalls, cryptography, backups, and DNS security.
- Tooling foundation with **Kali Linux** and later labs on **scanning, vulnerability assessment, and basic penetration testing**.
